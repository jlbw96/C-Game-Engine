Controls:
W = Forward
S = Backward
A = Turn Left
D = Turn Right
T = First Person
Y = Third Person
U = Switch Scene

Notes: 
-Switch scene will only load the second scene it will not loop back to the previous scene.
-Python script will generate the json file to the desktop, scale can be manually added to
 json file however is defaulted to [1,1,1] if json file does not include it.
- Json files are added in the assets/levels directory and then added in the Levels.txt
